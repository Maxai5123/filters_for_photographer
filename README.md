
# Проект "Фильтры для фото"

## Описание проекта
Данный проект создан для применения различных фильтров на фото.

## Цели проекта
Программа для применениия фильтров на фото.

## Используемые технологии
Python для создания программы для применения фильтров.

Также изпользуется встроенная библиотека argparse для использования аргументов, и pillow для работы с изображениями.

## Установка зависимостей
Для работы проекта у вас уже должен быть установлен Python3.
Загрузите на компьютер необходимые файлы, затем используйте `pip` для установки зависимостей и установить зависимости. Зависимости можно установить командой, представленной ниже.
Установите зависимости командой:
```
pip install -r requirements.txt
```

## Пример запуска
Если вы хотите указать путь к вашему изображению, необходимо записать:
```
python main.py --photo_name 'image/photo.jpg'
```
Этот аргумент является необязательным, то есть его можно не указывать

Если вы хотите использовать контрастный фильтр, необходимо записать:
```
python main.py --contrast
```

Если вы хотите использовать чёрно-белый фильтр, необходимо записать:
```
python main.py --black_white
```

Если вы хотите использовать заблюренный фильтр, необходимо записать:
```
python main.py --blur
```

Если вы хотите добавить рамку на фото, необходимо записать:
```
python main.py --frame
```

Если вы хотите использовать медианный фильтр, необходимо записать:
```
python main.py --median
```

Если вы хотите использовать фильтр сепии, необходимо записать:
```
python main.py --sepia
```

Если вы хотите использовать все фильтры, необходимо записать:
```
python main.py --sepia --median --frame --blur --black_white --contrast 
```
